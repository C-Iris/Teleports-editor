# Teleports-editor
将本程序放置在与作弊器相同的目录

如下图所示，在分类框中选好目标，下方出现蓝色字“读取到..个坐标文件”即可点击“写出点位”

# 自助转换功能
1. 将包含.json的文件夹拖入窗口内，即可自动识别路径。
2. 支持自定义输出文件夹。
3. 支持自定义输出名称。
4. 确认无误后，单击“转换”即可。
